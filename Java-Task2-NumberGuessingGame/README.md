# Task 2: Number Guessing Game (Java)

A console-based number guessing game developed as part of the Oasis Infobyte Internship (OIBSIP).

## Features Implemented
- [x] **Random Generation:** Generates a number dynamically per round using `java.util.Random`.
- [x] **Smart Hints:** Provides real-time feedback ("Too High!", "Too Low!").
- [x] **Attempt Counter & Caps:** Visually monitors attempts and caps them depending on mode.
- [x] **Multi-Round Tracking:** Summarizes attempts at the end of each round.
- [x] **Play Again:** Allows continuous rounds without restarting the program.
- [x] **Bonus (Difficulty Levels):** Easy, Medium, and Hard presets.

## How to Run
1. Open your terminal or command prompt.
2. Compile the file: `javac NumberGuessingGame.java`
3. Run the application: `java NumberGuessingGame`
